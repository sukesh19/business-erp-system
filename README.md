# NexusERP AI - Monitoring and Observability Stack

This repository provides the complete observability and alerting setup for **NexusERP AI**, powered by **Prometheus**, **Grafana**, and **Alertmanager**. It offers full visibility into system performance, AI agent behavior, application reliability, and user metrics.

---

## Overview

The monitoring stack includes:
- **Prometheus**: Metric collection and rule-based alerting  
- **Alertmanager**: Alert routing, notification, and escalation  
- **Grafana**: Real-time dashboards and visualizations  
- **Exporters**: Data sources for system, application, and database metrics

This setup helps proactively detect issues like high CPU usage, slow API response times, database or Redis failures, AI agent execution errors, and customer churn patterns.
